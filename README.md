# Resume website
Resume (cv) page for Danielle.

## Tools used
- Sanity [https://www.sanity.io]
- Gatsby [https://www.gatsbyjs.com]
- TypeScript
- GraphQL

## Local Development
Clone this repository and run `make`.

### Commands
```
make build
make start
make stop
make dev
make test
make update
make reset
```

### Requirements
- Docker
- Make
